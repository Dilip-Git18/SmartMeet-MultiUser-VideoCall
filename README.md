# SmartMeet

SmartMeet is a real-time multi-user video conferencing web application built with WebRTC. It allows users to create or join secure meetings using secret meeting keys and shareable invitation links. The platform supports live video and audio communication, meeting controls, real-time chat, and image sharing.

## Features

### Video Meetings

* Create and join meetings using a secret meeting key
* Share direct meeting invitation links
* Multi-user browser-based video calls using WebRTC mesh connections
* Mute or unmute the microphone
* Turn the camera on or off
* Automatic speaker focus
* Click a participant to pin their video tile

### Host Controls

* Mute all participant microphones
* Turn off all participant cameras
* Manage the meeting title
* Display the meeting title for all participants

### Chat and Media Sharing

* Real-time text messaging
* Share images during meetings
* Image-only file uploads
* Chat messages and shared images automatically expire after 24 hours

### Admin Panel

* Dedicated secure admin page
* View detailed information for each meeting
* Access meeting and participant details

## Technologies

* WebRTC
* Real-time communication technologies
* Web-based frontend and backend

## Project Purpose

SmartMeet provides a simple and secure platform for real-time online meetings. It combines multi-user video calling, host controls, live chat, image sharing, and meeting management in one application.
